## 🖥️ What hyperyap does

hyperyap turns your voice into text on Windows. It runs on your PC, so your audio stays local. You do not need cloud access or an internet connection.

It is built for quick dictation. You can use hotkeys, start it with Windows, and keep a terminal window ready for text input. It uses NVIDIA Parakeet TDT for speech to text.
